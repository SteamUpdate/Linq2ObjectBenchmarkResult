# `for` vs `foreach` with `List<T>`

## Code
```csharp
public void Common()
{
    foreach (var item in list)
    {
    }
}

public void Optimized()
{
    for (var i = 0; i < list.Count; i++)
    {
        var item = list[i];
    }
}
```
